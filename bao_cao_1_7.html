AI RAG Chatbot
for Food Recommendations

Hệ thống gợi ý ẩm thực thông minh với RAG, Angular và .NET

  • Công nghệ lõi: Áp dụng RAG (Retrieval-Augmented Generation) kết hợp LLM.
  • Kiến trúc: Frontend Angular 21, Backend .NET 10, AI Engine Python FastAPI.
  • Triển khai: Container hóa toàn diện, chạy dễ dàng với Docker Compose.

Trải nghiệm & Tính năng chính

Người dùng (Client)

  • Gợi ý món ăn thông minh qua chat Text/Voice.
  • Lưu trữ, ghim (pin) các cuộc hội thoại quan trọng.
  • Đánh giá (Like/Dislike) và báo cáo tin nhắn rác.

Quản trị viên (Admin)

  • Dashboard thống kê người dùng và tin nhắn.
  • Theo dõi biểu đồ tương tác, xét duyệt report.
  • Xuất báo cáo thống kê dưới dạng CSV.

Kiến trúc: Microservices & Docker

  • Container hóa: 4 cụm services độc lập: DB, Python AI, .NET API, Angular UI.
  • Docker Network: Giao tiếp nội bộ an toàn qua chatbot_network.
  • Tối ưu AI Host: Lấy LLM từ Host (Ollama) thay vì ôm vào container, giúp tiết kiệm hàng chục GB dung lượng.

Khởi động 1 chạm

Chỉ cần docker-compose up -d để vận hành toàn bộ hệ thống

.NET Backend: Models & EF Core

Database Design

Sử dụng Entity Framework Core Code-First.

Cấu hình Fluent API chuyên sâu: Cascade Delete cho User, Restrict/SetNull cho Report để bảo toàn dữ liệu lịch sử.

Proxy Pattern & Mock Data

ChatController: Đóng vai trò Proxy điều phối request tới AI Python, đồng bộ hóa lưu trữ DB giữa user và AI.
AdminController: Cung cấp endpoint tự sinh dữ liệu giả (Mocking) phục vụ test biểu đồ Admin UI.

Angular 21: Kiến trúc Frontend

Xây dựng SPA mượt mà với tư duy Component-Based.

Smart Component (Root)

  • Đóng vai trò Orchestrator (Điều phối).
  • Truyền state xuống các component con qua @Input / @Output.
  • Quản lý vòng đời chặt chẽ với RxJS Subscriptions.

Optimistic UI

  • "Bơm" ngay tin nhắn tạm thời vào UI khi user gửi.
  • Hiệu ứng phản hồi tức thì dù API đang xử lý.
  • Nâng tầm trải nghiệm người dùng (UX) cho ứng dụng Chat.

Python AI: Kiến trúc RAG Pipeline

Retrieval (Tìm kiếm)

Sử dụng FAISS Vector DB lưu trữ tri thức. Lấy Top-K tài liệu liên quan nhất theo ngữ cảnh câu hỏi.

Embeddings & Fallback

Dùng mô hình nhúng tiếng Việt. Tự động chuyển qua paraphrase-multilingual-MiniLM-L12-v2 dự phòng hoặc build lại FAISS nếu thiếu file.

Generation (Sinh Text)

Gemma 4 (E2B) thông qua Ollama. Kết hợp Prompt Template để sinh câu trả lời chính xác, hạn chế Hallucination.

Tinh chỉnh: Finetune & Đánh giá

Fine-tuning Model

  • Tinh chỉnh Embedding Model riêng cho ẩm thực.
  • Sử dụng dữ liệu QA Dataset chuyên sâu.
  • Tăng độ chính xác khi truy vấn từ khóa đặc thù tiếng Việt.

Evaluate Cơ Chế Tìm Kiếm

  • Đo đạc bằng thuật toán khớp chuỗi mờ (Fuzzy Matching).
  • Tính toán Precision, Recall, F2-Score.
  • Đảm bảo FAISS luôn trả về dữ liệu chất lượng cao.

Tổng kết: Key Takeaways

Hệ thống AI RAG Hoàn chỉnh

Kiến trúc RAG tiêu chuẩn kết hợp với LLM tiên tiến, xử lý ngữ cảnh mượt mà.

Sự kết hợp .NET và Python

Khai thác sức mạnh nghiệp vụ/bảo mật của C# .NET và khả năng xử lý AI vô song của Python.

UI/UX Hiện đại, Triển khai dễ dàng

Angular SPA tối ưu trải nghiệm thời gian thực. Tất cả được đóng gói gọn nhẹ qua Docker Compose.

Đánh giá Hiệu năng: Benchmark Results

  • Thời gian phản hồi: Trung bình ~15.7s, nhanh nhất 5.9s (Chạy cục bộ CPU).

Hướng tối ưu tương lai

Sử dụng mô hình lớn hơn (ví dụ: Llama 3 8B) hoặc Fine-tune riêng để cải thiện khả năng tuân thủ định dạng Prompt.
